# Living Sphere Renderer v1 reference captures

These PNGs are deterministic native reference captures generated by
`scripts/living-sphere-snapshots.swift`. The harness mirrors the renderer's cached
geometry and evidence-neutral theme tokens without mounting a preview route in the
product UI or changing navigation.

| File | Configuration | Pixels |
| --- | --- | --- |
| `iphone-se-light.png` | iPhone SE, light appearance | 750 × 1334 |
| `iphone-17-pro-dark.png` | iPhone 17 Pro, dark appearance | 1206 × 2622 |
| `light-mode.png` | Standard reference, light appearance | 1179 × 2556 |
| `dark-mode.png` | Standard reference, dark appearance | 1179 × 2556 |
| `reduce-motion.png` | Standard reference, dark, static phase | 1179 × 2556 |

These captures verify composition and palette. Final FPS, energy, VoiceOver, and
device-specific GPU validation still belong in the real-screen integration phase.
