The file must export exactly these items in this order:

1. `NOTIFICATION_PREFS_KEY = '@vitalspan_notification_prefs'` — exported const (module-scope)

2. `NotificationPrefs` interface:
   ```
   { morning: { enabled: boolean; time: string }; afternoon: ...; evening: ...; night: ...; }
   ```
   where `time` is a 24h "HH:MM" string.

3. `DEFAULT_PREFS: NotificationPrefs` const with values: morning '08:00' disabled, afternoon '13:00' disabled, evening '18:00' disabled, night '21:00' disabled. All `enabled: false` by default.

4. `loadNotificationPrefs(): Promise<NotificationPrefs>` — reads NOTIFICATION_PREFS_KEY from AsyncStorage; returns parsed result typed as NotificationPrefs on success, DEFAULT_PREFS on error or missing key.

5. `saveNotificationPrefs(prefs: NotificationPrefs): Promise<void>` — writes JSON.stringify(prefs) to NOTIFICATION_PREFS_KEY; swallows errors silently.

6. `ensurePermission(): Promise<boolean>` — calls `Notifications.getPermissionsAsync()` first; if status is already 'granted', returns true immediately without prompting again. Otherwise calls `Notifications.requestPermissionsAsync()` and returns `status === 'granted'`. Wraps in try/catch returning false on error.

7. `scheduleSlot(slot: TimeSlot, time: string): Promise<void>` — calls `Notifications.scheduleNotificationAsync` with:
   - identifier: `vitalspan-${slot}` (e.g., "vitalspan-morning")
   - content: `{ title: 'Vitalspan Reminder', body: 'Time to take your ${slot} supplements.', sound: true }`
   - trigger: `{ type: Notifications.SchedulableTriggerInputTypes.DAILY, hour: number, minute: number }`
   Parse hour/minute from time string via `time.split(':').map(Number)`.
   Use `SchedulableTriggerInputTypes.DAILY` (NOT CalendarTriggerInput).

8. `cancelSlot(slot: TimeSlot): Promise<void>` — calls `Notifications.cancelScheduledNotificationAsync('vitalspan-${slot}')`.

9. `rescheduleAll(prefs: NotificationPrefs): Promise<void>` — calls `Notifications.cancelAllScheduledNotificationsAsync()` first, then loops over `['morning', 'afternoon', 'evening', 'night'] as TimeSlot[]` and calls `scheduleSlot` for each slot where `prefs[slot].enabled === true`.

TypeScript rules: no `any`; all parameters and return types explicit. No default export.
