import Svg, { Circle, Line, Path, Ellipse } from 'react-native-svg';

<!-- From src/theme/index.ts — use these exact token paths -->
Colors.accent: '#5B9DBF'       — all figure dots and lines
Colors.Beige.bgShade: '#E4E0D4' — (available but figures should be on transparent bg)

<!-- Component contract: each exercise illustration component -->
Props: { size?: number }  — width and height in pixels; default 120
Returns: <Svg viewBox="0 0 100 100" width={size} height={size}>...</Svg>
No state, no animation (D-03: static figures), no imports except React and react-native-svg + Colors

<!-- SVG geometry guide — keep each component under ~1KB -->
A neural-dot human figure consists of:
1. Body dots: 6-12 Circle elements, r=2-3, fill=Colors.accent, opacity=0.7-0.9
   Positions: head (50,10), neck (50,18), shoulders (30,25) and (70,25), 
   hips (35,55) and (65,55), knees (35,72) and (65,72), ankles (35,88) and (65,88)
   (adjust for the specific exercise position)
2. Connecting lines: Line elements between adjacent dots, stroke=Colors.accent, strokeWidth=1.5, opacity=0.5
3. Optional movement arrow: 2-3 Line or Path elements showing movement direction, opacity=0.6

Exercise-specific figure variations — adapt the base skeleton:
- Standing (default): upright, dots at standard positions above
- Squatting (squats, lunges): knees bent — lower knee/ankle dots, hips lower
- Lying down (crunches, sit-ups, stretches): rotate entire figure 90 degrees, dots horizontal
- Bent-over rows: torso angled ~45 deg, dots shifted accordingly
- Pressing (push movements): arms extended forward/up, extra arm dots
- Pulling (rows, chins): arms bent, torso angled
- Cardio/jumping: legs spread or one leg forward
- Plank/bridge: fully horizontal, arms extended

Movement arrows: for exercises with clear direction of movement, add 2-3 Line elements as a simple arrow (two short lines meeting at a point) showing the direction (up, down, lateral, rotational). Color: Colors.accent, opacity 0.8.

Each file exports a single default component. Component name matches the filename (PascalCase of the illustrationId).
Example file structure (for plank.tsx):
  import React from 'react';
  import Svg, { Circle, Line } from 'react-native-svg';
  import { Colors } from '../../theme';

  interface Props { size?: number }
  export default function Plank({ size = 120 }: Props) {
    return (
      <Svg viewBox="0 0 100 100" width={size} height={size}>
        [SVG elements — dots and lines for plank position]
      </Svg>
    );
  }

Keep each file to ~20-35 lines. Total SVG elements per figure: 8-20 elements. Do NOT create complex anatomy.
