    TYPE: `type Nav = NativeStackNavigationProp<RootStackParamList>;`

    STATE:
    - `sheet: 'none' | 'signup' | 'login'` — which bottom sheet is visible
    - `email: string`, `password: string`, `confirmPassword: string` (signup only)
    - `loading: boolean` — auth request in flight
    - `error: string | null` — inline error message
    - `metricValue: Animated.Value` — for the animated longevity metric preview (initialized to 0)

    SCREEN HEIGHT: `const { height: SCREEN_H } = Dimensions.get('window');`

    BOTTOM SHEET ANIMATION:
    - `const sheetAnim = useRef(new Animated.Value(SCREEN_H)).current`
    - `function openSheet(type: 'signup' | 'login')`:
      - `setSheet(type); setError(null); setEmail(''); setPassword(''); setConfirmPassword('');`
      - `Animated.timing(sheetAnim, { toValue: 0, duration: 350, useNativeDriver: true }).start()`
    - `function closeSheet()`:
      - `Animated.timing(sheetAnim, { toValue: SCREEN_H, duration: 300, useNativeDriver: true }).start(() => setSheet('none'))`

    ANIMATED METRIC PREVIEW (D-02):
    - On mount, useEffect: `Animated.loop(Animated.sequence([Animated.timing(metricValue, { toValue: 1, duration: 3000, useNativeDriver: true }), Animated.timing(metricValue, { toValue: 0, duration: 3000, useNativeDriver: true })]), { iterations: -1 }).start()`
    - Render a row of 3 small metric "orbs" (View circles) with interpolated opacity/scale driven by metricValue — this is the subtle animated preview behind the CTAs. Keep it visually subtle (opacity 0.3→0.7). Display labels: "HRV", "Glucose", "BioAge" in small muted text below each orb.

    SIGN UP HANDLER (`handleSignUp`):
    1. Validate: email non-empty, password >= 8 chars, confirmPassword matches. Set error if invalid. Return early.
    2. `setLoading(true); setError(null);`
    3. Get current user: `const { data: { user: currentUser } } = await supabase.auth.getUser()`
    4. If `currentUser?.is_anonymous`:
       - Call `convertAnonymousToEmail(email, password)` (promotes anonymous session to email account).
       - On success: trigger migration with `@vitalspan_identity_linked` idempotency guard (read flag → if not set → call migrateHistory → set flag 'true').
    5. Else: call `signUpWithEmail(email, password)`.
    6. If result.error: `setError(result.error); setLoading(false); return;`
    7. On success: close sheet, navigate to SignUpConfirmation with email: `nav.navigate('SignUpConfirmation', { email })`
    8. Wrap entire handler in try/catch; catch sets `setError(mapAuthError(e instanceof Error ? e.message : String(e)))`.

    LOGIN HANDLER (`handleLogin`):
    1. Validate: email non-empty, password non-empty.
    2. `setLoading(true); setError(null);`
    3. Call `signInWithEmail(email, password)`.
    4. If result.error: `setError(result.error); setLoading(false); return;`
    5. On success: check onboarding:
       - `const raw = await AsyncStorage.getItem('@vitalspan_user_profile').catch(() => null)`
       - `const profile = raw ? JSON.parse(raw) : null`
       - `if (profile?.onboardingComplete) { nav.reset({ index: 0, routes: [{ name: 'Main' }] }) } else { nav.reset({ index: 0, routes: [{ name: 'Onboarding' }] }) }`
    6. Wrap in try/catch.

    CONTINUE AS GUEST HANDLER (`handleGuest`) — per D-11:
    - `const raw = await AsyncStorage.getItem('@vitalspan_user_profile').catch(() => null)`
    - `const profile = raw ? JSON.parse(raw) : null`
    - `if (profile?.onboardingComplete) { nav.reset({ index: 0, routes: [{ name: 'Main' }] }) } else { nav.reset({ index: 0, routes: [{ name: 'Onboarding' }] }) }`
